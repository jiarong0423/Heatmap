## 🔗 使用說明

### 💡 投資決策流程

**步驟 1️⃣：監控美股板塊**
1. 觀察上方實時行情表格
2. 識別資金流入/流出的板塊
3. 注意漲跌幅超過 2% 的強勢板塊

**步驟 2️⃣：找出對應族群**
1. 使用對應表查詢該板塊對應的台股族群
2. 一個美股板塊可能對應多個台股族群

**步驟 3️⃣：執行操作**
1. 資金流入（上漲）→ 考慮買進相關個股
2. 資金流出（下跌）→ 考慮回避或減碼

### 📊 圖標說明

**資金流入（上漲）：**
- 🚀 強勢上漲 (漲幅 ≥ 2%) - **強買信號！**
- 📈 上漲 (漲幅 1% ~ 2%) - 溫和上漲，關注機會
- ➡️ 平穩上漲 (漲幅 0% ~ 1%) - 橫盤整理

**資金流出（下跌）：**
- ⛔ 強勢下跌 (跌幅 ≥ 2%) - **強賣信號！**
- 📉 下跌 (跌幅 1% ~ 2%) - 溫和下跌，等待反彈
- ⚠️ 平穩下跌 (跌幅 0% ~ 1%) - 輕微下跌，觀望
